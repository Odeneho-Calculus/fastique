About Fastique

Instant File System Search Utility

Fastique is a high-performance desktop search utility built for lightning-fast file discovery and manipulation. Designed with a sleek web-based interface powered by Flask and styled with modern HTML/CSS, it offers real-time results and seamless file operationsâ€”all while keeping the core logic in Python.

Key Features

Instant Search

  • Live results as you type
  • Smart filtering by file type, date, and size
  • Advanced options including regex and case sensitivity

Built-In File Operations

  • Open, copy, move, delete, and rename files
  • Create new folders and files
  • View detailed file metadata

Modern Web UI

  • Responsive design with smooth animations
  • Search bar with auto-suggestions
  • Multiple view options for results

Performance Focused

  • Threaded search to keep UI responsive
  • Smart indexing for frequent paths
  • Memory and CPU efficiency

Technology Stack

  • Backend: Python, Flask
  • Core Libraries: os, shutil, subprocess, threading
  • Frontend: HTML5, CSS3, JavaScript
  • Architecture: Modular design with threading for performance

Current Version: 1.0
